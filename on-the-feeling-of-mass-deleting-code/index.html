On the feeling of mass-deleting code

September 2025 · 1 min read

I deleted 2,400 lines today. It felt like spring cleaning.

The code had been dead for months — behind a feature flag that was never turned on, calling an API that no longer existed. But it sat there, passing CI, showing up in search results, confusing anyone who stumbled into it.

Why we don't delete

Fear, mostly. "Someone might need this." But version control exists. The code isn't gone — it's just not in the way anymore.
